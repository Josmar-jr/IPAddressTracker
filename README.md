## 📀 How To Use

```bash
# Clone this repository
$ git clone https://github.com/Josmar-jr/IPAddressTracker.git

# Go into the repository
$ cd IPAddressTracker

# Install dependencies
$ yarn install

# Run the app
$ yarn start
```
